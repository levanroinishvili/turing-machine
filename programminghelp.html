Post Automaton

The Post Automaton consists of a Film Strip and a Cursor. The Film Strip consists of frames, each of which can be either empty or full. The Cursos can move along the Film Strip and execute directives. Directives are numbered sequentially and written as a program.


Pic 1. Film Strip and the Cursor of a Post Automaton

After you write the program, use the buttons under the Film Strip to execute and debug the program.

You can also export and import programs, and view this help by using these buttons.

Programming Directives

Short Name   Full Name   Description
L   LEFT   Move the Cursor left
R   RIGHT   Move the Cursor right
E   EVALUATE   Evaluate the frame under cursor.
1       Write to the frame under the Cursor. If the cell is already written, error occurs.
0       Empty the frame under the Cursos. If the cell is already empty, error occurs.
S   STOP   Stop program
